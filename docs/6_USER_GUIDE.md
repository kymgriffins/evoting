# 👤 USER GUIDE — How to Use the App (by Role)

---

## 👑 Admin Guide (5 accounts available)

### What admins can do:
- Create elections
- Add positions to elections (President, VP, Secretary, etc.)
- Approve or reject candidate registrations
- View audit logs (every action is logged)
- See system-wide statistics

### Step-by-Step:

**1. Create an Election**
```
Login as admin → Dashboard → "Manage Elections"
→ Fill in title, description, start/end dates
→ Check "Active" to make it live
→ Click "Create Election"
```

**2. Add Positions**
```
Click "Positions" next to the election → "Add Position"
→ Title: "President", Max Votes: 1
→ Repeat for VP, Secretary, Treasurer
```

**3. Approve Candidates**
```
Dashboard → "Manage Candidates"
→ See list of pending candidates
→ Click "Approve" or "Reject"
```

**4. View Audit Logs**
```
Dashboard → "Audit Logs"
→ See every action: who voted, who registered, who was approved
```

## 🗣️ Candidate Guide (10 accounts available)

### What candidates can do:
- Upload manifestos (campaign promises) in 5 categories
- Edit/delete their manifestos
- Post progress updates after election (Not Started → In Progress → Completed → Delayed)
- View their ratings from voters

### Step-by-Step:

**1. Login as a candidate**
```
cand001 / cand123
```

**2. Create Manifestos**
```
Dashboard → "My Manifestos" → "+ Add Manifesto"
→ Title: "Campus Wi-Fi Upgrade"
→ Description: Explain your promise in detail
→ Category: Pick one (Education, Healthcare, etc.)
→ Click "Create Manifesto"
→ Repeat 5 times per candidate (we pre-loaded 5 each)
```

**3. Post Progress Updates**
```
Dashboard → "My Manifestos"
→ Click "Updates" on a manifesto
→ Select status: Completed / In Progress / Delayed / Not Started
→ Add description of what was done
→ Optionally add evidence URL (link to a report)
→ Click "Post Update"
```

**4. View Your Performance**
```
Navigation → "Leaderboard"
→ See your rank, success rate, and average star rating
```

## 🗳️ Voter Guide (600 accounts available)

### What voters can do:
- Vote in active elections (one vote per position)
- Rate manifestos (1-5 stars + comments)
- View election results
- View manifesto tracking (what candidates actually delivered)
- View leaderboard

### Step-by-Step:

**1. Login as a voter**
```
v0001 / voter123
```

**2. Vote in an Election**
```
Dashboard → Active Elections → "Cast Your Vote"
→ For each position, select one candidate
→ Click "Submit My Votes"
→ You cannot vote again for the same position
```

**3. Rate Manifestos**
```
Navigation → "Tracking"
→ Find a candidate → Click "Rate" on any manifesto
→ Select 1-5 stars (click the star)
→ Optionally write a comment
→ Click "Submit Rating"
```
Or go to a specific manifesto page:
```
Navigation → "Tracking" → Click candidate name
→ Click "Updates" on a manifesto
→ Rate directly on the detail page
```

**4. View Results**
```
Navigation → "Results"
→ See past election results with vote counts
```

**5. View Leaderboard**
```
Navigation → "Leaderboard"
→ See which candidates fulfilled the most promises
→ Ranked by success rate (% completed)
→ Top 3 get 🥇🥈🥉
